Assistant analytique RAG pour l’analyse de performance sportive

Conception d’un assistant intelligent combinant Retrieval-Augmented Generation (RAG) et interrogation SQL pour analyser les performances NBA et contextualiser les statistiques. Architecture hybride de type routing intelligent (RAG / SQL / HYBRID) permettant d’adapter dynamiquement la stratégie de réponse selon la requête utilisateur. Le système s’appuie sur PostgreSQL, FAISS, Mistral, LangChain, Pydantic et une évaluation via RAGAS pour améliorer la fiabilité des réponses et réduire les hallucinations. Le projet met en évidence les limites d’une architecture RAG classique et les gains apportés par un routing intelligent.

Faithfulness : 1.00 (SQL) · 0.98 (RAG) · 0.79 (Hybrid)
Context Recall : jusqu’à 0.76 (Hybrid)
RAG LLM SQL LangChain Mistral FAISS PostgreSQL RAGAS

Détection de tumeurs cérébrales sur IRM

Projet de deep learning appliqué au computer vision pour l’imagerie médicale, visant la détection de tumeurs cérébrales à partir d’IRM dans un contexte de données fortement déséquilibrées et partiellement labellisées. Comparaison d’approches supervisée, non supervisée et semi-supervisée avec extraction de features via ResNet50, clustering et CNN custom. Les résultats montrent l’intérêt de l’exploitation des données non annotées, avec une amélioration nette des performances par rapport au supervisé pur.
